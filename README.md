## 📜 Script Descriptions

### 🖥️ Script 1 — System Identity Report

Displays system details such as kernel, user, uptime, and Linux distribution.

### 📦 Script 2 — Package Inspector

Checks if Python is installed and displays version details.

### 💾 Script 3 — Disk & Permission Auditor

Analyzes important system directories and shows permissions and disk usage.

### 📊 Script 4 — Log File Analyzer

Reads a log file, counts keyword occurrences, and displays recent matches.

### 🧾 Script 5 — Open Source Manifesto Generator

Takes user input and generates a personalized open-source statement.
